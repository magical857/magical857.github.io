记录一下中午markDown语法的学习

学习的网站Markdown Tutorial

1. 斜字体，字体加粗强调

  •     在需要变斜字体或词语左右加"_"或者"*",例如：_wewe_、*dwdw*

  •     在需要变粗的的字体或词语左右加"**",例如:**dwdw**

2. 标题（最多六级）

  • 一级标题

  • 二级标题

  • 三级标题

  • 四级标题

  • 五级标题
  • 六级标题

3. 链接（图片和文字）

  • 第一种
    在[ ]添加描述文本,紧接着后面加上（ ），圆括号里面放链接例如：[这是一个例子]（https://siguso.com/）
    效果：这是一个例子
  • 第二种
    全局链接
    例如:
 这是 [其它玩意][another place].  
 这是 [另一个链接][another-link].  
 现在回到 [第一个链接][another place].

 [another place]: www.github.com  
 [another-link]: www.bing.com

效果如下：
这是 其它玩意.
这是 另一个链接.
现在回到 第一个链接.

创建一个内联的图片链接,需要输入一个感叹号!将方括号 [ ]中的描述性文本包裹起来， 然后使用圆括号 ( )包裹链接地址。https://upload.wikimedia.org/wikipedia/commons/5/56/Tiger.50.jpg
